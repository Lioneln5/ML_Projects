Liver Cirrhosis Prediction Model

## Overview
This project explores the use of machine learning to predict liver cirrhosis outcomes based on patient health data. The goal is to identify key factors associated with disease progression and evaluate how well different features can predict patient status.

## Dataset
The dataset contains patient-level medical information:
- Age, sex
- Laboratory test results
- Clinical indicators

The data was cleaned and preprocessed to handle missing values and prepare it for modeling.

## Approach
The workflow for this project included:
1. Data cleaning and preprocessing
2. Feature selection and encoding
3. Model training using machine learning techniques
4. Model evaluation following standard metrics

## Results
The model was able to identify patterns in the data and achieve reasonable predictive performance. Certain medical features showed stronger influence on predictions, indicating their importance in disease progression.

## Key Takeaways
- Data preprocessing plays a crucial role in model performance  
- Some features contribute significantly more to predictions than others  
- The model has limitations due to the dataset size and feature scope
